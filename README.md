## Script "StudyMoney"

Este é um programa em Python executado via terminal que auxilia no controle financeiro pessoal focado no cumprimento de um objetivo de compra.

🛠️ Funcionalidades Principais:
* **Definição de Meta:** Permite estipular o objetivo de compra e o valor total necessário.
* **Menu Interativo (Loop):** Roda continuamente apresentando as seguintes opções ao usuário:
  1. **Registrar Entrada:** Adição de valores ao saldo (salário, mesada, etc.).
  2. **Registrar Gasto:** Dedução de valores com validação de saldo dinâmico.
  3. **Status da Meta:** Exibição do saldo atualizado e o progresso em porcentagem para atingir o objetivo.
  4. **Sair:** Encerramento do programa.
* **Histórico Simplificado:** Armazenamento dos gastos em estrutura de lista para exibição de extrato.
